General Information

Support for Bro logging library.

Bro, a powerful network security monitor, which by default churns out ASCII logs in a easily parseable whitespace separated (column) format from network traffic, live or PCAP.

This toolkit provides multiple BroParser composites with a ready to use pre-configured AdaptiveParser operator, parameterized for Bro format parsing.
