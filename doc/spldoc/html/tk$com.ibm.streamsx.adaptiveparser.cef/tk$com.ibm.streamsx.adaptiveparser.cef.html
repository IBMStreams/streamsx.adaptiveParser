General Information

Support for Common Event Format (CEF) library.

The ArcSight Common Event Format defines a syslog based event format that addresses the need to define core fields for event correlation for all vendors integrating with ArcSight.

This toolkit provides CEFParser composite with a ready to use pre-configured AdaptiveParser operator, parameterized for CEF format parsing.
