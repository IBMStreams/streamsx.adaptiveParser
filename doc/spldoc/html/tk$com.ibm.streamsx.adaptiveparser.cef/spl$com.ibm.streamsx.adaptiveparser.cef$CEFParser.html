composite CEFParser(output CEFEvent; input RawData)

CEFParser composite uses pre-configured AdaptiveParser operator, parameterized for CEF format parsing. As an option, CEF header can be represented as list<rstring>[7] instead of tuple

Parameters

  • dataAttr: an input attribute name which holds data to parse
  • batch: expect multiple events when parsing the data

Input Ports

  • RawData: blob or rstring data to be parsed

Output Ports

  • CEFEvent: CEFParser.CEFHeader_t , CEFParser.CEFExtension_t
